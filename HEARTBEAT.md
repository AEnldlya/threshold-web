# HEARTBEAT.md - Daily Verified Business Discovery & Building

## Primary Daily Task - Every Morning at 7 AM ET

**VERIFICATION PHASE:**
1. Find 20-30 new Boston/area businesses (Google Maps, Yelp, directories)
2. Verify each one has NO website:
   - Google search "[Business Name] website"
   - Check social media for website links
   - Confirm NO website exists
3. Build call list with only verified NO-website businesses
4. Send to user: "Today's 15 verified businesses" with names, phone numbers, category

**CALLING PHASE (User does this):**
1. User calls 15 verified businesses
2. User reports back: "YES to: [Business Name] (phone)"
3. I log to tracker
4. **When user says "build [Business Name]"**: Start professional website build

## Build Process (Quality > Speed)

When user confirms "build [Business]":
- **Day 1-2**: Discovery call + Figma design mockup (user approves)
- **Day 3-5**: Development (Next.js, TypeScript, Tailwind, Framer Motion animations)
- **Day 6-7**: Refinement (Lighthouse 95+, WCAG AA accessibility, SEO, structured data)
- **Day 8-9**: Deploy to staging, user reviews, revisions
- **Day 10**: Launch to production + Google Business setup

**Price: $2,500** (minimum, based on quality)
**Quality Standards:**
- Lighthouse score: 95+
- WCAG AA accessibility compliance
- WebP/AVIF images optimized
- Custom animations (Framer Motion)
- Mobile-first responsive
- SEO optimized
- No frameworks, clean architecture

## Call Script (for user)
"Hi, is the owner or manager available? This is Andy. I noticed you don't have a website and we just built one for [similar business in same category]. Interested in seeing a quick example? Takes 2 minutes."

## When User Says "build [business name]"
Drop everything and create professional website:
- Professional design (Next.js + Tailwind)
- Responsive mobile/tablet/desktop
- Optimized images (WebP/AVIF)
- Fast loading (Lighthouse 90+)
- Ready to deploy

## Tracking
- Log to: DAILY_CALL_RESULTS.csv
- Track: Business name, phone, date called, YES/NO

## Success Metrics
- 15 calls/day = 75 calls/week
- Target: 10-15 interested/week
- Target: 4-6 closed/week = $2000-3000/week
- All businesses verified to have NO website (high relevance)
