WHAT HAPPENS AT 7 AM
================================================================================

1. Cron job triggers daily_prospect_report.py
2. Script loads 167 Boston companies
3. Checks sent_emails_log.json (what's already sent)
4. Selects 50 unsent companies with highest confidence
5. Groups by category (restaurants, salons, plumbing, etc.)
6. Loads daily_responses.json (what's coming back)
7. Formats beautiful report with all details
8. Saves to daily_report.txt
9. Saves email list to pending_send_batch.json
10. Notifies you (via message/log)
11. Waits for your approval

CUSTOMIZATION OPTIONS
================================================================================

Change Time (Default: 7 AM UTC):
  Edit cron command, change "0 7" to different hour
  Example: "0 9" = 9 AM UTC

Change Prospect Count (Default: 50):
  Edit daily_prospect_report.py, line ~80
  Change: return unsent[:50]
  To:     return unsent[:25] or return unsent[:100]

Change Email Delay (Default: 5 minutes):
  Edit send_approved_batch.py, line ~18
  Change: DELAY_SECONDS = 300  (5*60)
  To:     DELAY_SECONDS = 180  (3*60) or 600 (10*60)

Change Email Template:
  Update EMAIL_TEMPLATES_HUMANIZED.md
  Or create new template file
  Edit send_approved_batch.py to load it
