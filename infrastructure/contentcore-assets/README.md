## Export Specifications

### Video Settings
- **Format:** WEBM (VP9 codec)
- **Resolution:** 4K (3840×2160) for flexibility
- **Frame Rate:** 24fps (cinematic feel)
- **Transparency:** Alpha channel enabled
- **Compression:** Quality 80%, file size <5MB per asset
- **Duration:** 6-10 seconds loop

### Delivery Format
```html
<video 
  autoplay 
  loop 
  muted 
  playsinline
  style="mix-blend-mode: screen;"
>
  <source src="asset.webm" type="video/webm">
</video>
```

### CSS Integration
```css
.video-3d {
  position: absolute;
  z-index: 1;
  mix-blend-mode: screen; /* or multiply, overlay */
  pointer-events: none;
  width: 100%;
  height: auto;
}
```

## Usage Guidelines

### Hero Section
- Use device mockup at 60-80% viewport width
- Position: center or offset based on text layout
- Blend mode: screen for dark backgrounds, multiply for light

### Feature Cards
- Use abstract shapes at 100-200px size
- Position: floating beside or behind text
- Animation: subtle rotation or pulse

### About Section
- Use logotype 3D treatment
- Position: background element, low opacity
- Animation: slow rotation or drift

### Product Pages
- Use rotating platform or exploded view
- Position: hero focal point
- Animation: continuous rotation
