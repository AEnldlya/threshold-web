## Key Lessons Learned
- Emojis as icons = amateur. Use inline SVGs.
- CSS custom properties at :root make everything consistent and easy to change
- HSL colors are easier to manipulate than hex
- clamp() for type/spacing = fewer media queries
- auto-fit/minmax in Grid = self-responsive layouts
- text-wrap: balance on headings = professional look
- LocalBusiness JSON-LD schema = important for Google rich results
- Phone number as tel: link = critical for mobile users
- Netlify `netlify` attribute on forms = free form backend, no code needed
- Hosting: Netlify/GitHub Pages = free, drag-and-drop deploy
