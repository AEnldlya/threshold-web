## YOUR 3 E-COMMERCE OPTIONS

### Option 1: STRIPE PAYMENT LINKS (Easiest & Best)

**What it is:** Simple links you create in Stripe Dashboard. Customer clicks → buys → done.

**Cost:** 2.9% + $0.30 per transaction
- On $100 sale: $3 fee
- On $500 sale: $15 fee

**Setup time:** 2 minutes per product

**Best for:** Quick upsell, simple products (5-20 items max)

**How to integrate:**
- Create product in Stripe Dashboard
- Get payment link (e.g., buy.stripe.com/test_ABC123)
- Add to website as button/link
- Takes 15 min to set up on website

**Example use:**
```
[Client runs salon]
You create link: buy.stripe.com/client-shop
Add to website: "Shop Our Products" button → links to Stripe page
Customer clicks → sees products → buys
Money goes to client's Stripe account
```

**Pitch:** 
```
"A lot of salons/restaurants sell products or gift cards online. 
I can add a simple 'Shop' page that links to a payment system. 
Takes me 2 hours, costs you $100. You keep everything except 
a small 2.9% + $0.30 fee per sale."
```

---

### Option 2: SNIPCART (Full Shopping Cart)

**What it is:** Embedded shopping cart on your website (like a real store)

**Cost:** 2% of sales (or $20/month if under $1000/month sales)

**Setup time:** 3-4 hours to set up properly

**Best for:** Serious e-commerce (50+ products, inventory tracking)

**Features:**
- Full shopping cart
- Inventory management
- Abandoned cart emails
- Customer accounts
- Multi-currency support

**How to integrate:**
- Add code snippet to website
- Set up products in Snipcart Dashboard
- Customer clicks → cart icon → checkout
- Looks professional, feels like real store

**Example use:**
```
[Client runs boutique with 40 products]
You set up Snipcart with their inventory
Customer browses → adds to cart → checks out
Full shopping cart experience
Client gets $$ minus 2% fee
```

**Pitch:**
```
"If you want to sell products online, I can add a full 
shopping cart to your site. Customers can browse, add to cart, 
checkout. I set it up, you manage the inventory. 
Takes me 4 hours, costs $200-300. You pay 2% per sale."
```

---

### Option 3: SQUARE ONLINE (All-in-one if they use Square)

**What it is:** If client already has Square (POS system), they can add Square Online

**Cost:** 2.6% + 30¢ + payment processing

**Best for:** Clients who already use Square for in-store payments

**How to integrate:**
- They set it up in their Square Dashboard
- Links to their existing Square account
- You just embed link/button on site

**Pitch:**
```
"I noticed you use Square in-store. Square has an online 
store feature that syncs with your register. I can add it to 
your site. They handle everything. You just manage inventory 
the same way you do in-store."
```

## TECHNICAL SETUP (For You)

### Stripe Payment Links Setup (15 min)
1. Client signs up for Stripe account
2. You create product in Stripe Dashboard
3. Get shareable link
4. Add button to website pointing to link
5. Done

### Snipcart Setup (3-4 hours)
1. Client signs up for Snipcart account
2. You add Snipcart code snippet to website
3. Client lists products in Snipcart Dashboard
4. You set up payment processing
5. Test checkout flow
6. Train client on inventory management
7. Done

### Integration on v4/v5 Template
Easy because both are static HTML:
- Add button: `<a href="https://buy.stripe.com/..." class="btn">Shop Now</a>`
- Or embed Snipcart cart code in footer
- Responsive on mobile automatically
