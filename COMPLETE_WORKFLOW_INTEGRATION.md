    ├─ Direct domain check (.com, .net, etc.)

    "Today's 15 verified prospects ready for calls"

You review prospects

## Troubleshooting & Support

### Common Issues

**Issue**: Auto-Deploy Bot hangs on approval  
**Solution**: Check if response was received. Retry with "yes" or "no"

**Issue**: Domain not resolving after purchase  
**Solution**: DNS propagation takes 5-30 minutes. Wait and retry.

**Issue**: Website not deploying to Vercel  
**Solution**: Check GitHub connection. Verify Vercel API key is valid.

**Issue**: Stripe payment not processed  
**Solution**: Check Stripe API keys. Verify customer card is valid.
