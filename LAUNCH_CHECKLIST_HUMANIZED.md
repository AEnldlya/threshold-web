## Real Talk: Failure Points

**Emails bounce:**
- Normal rate: 5-10%
- If higher: your estimated emails might be slightly off
- Fix: Call the business, get the right email, try again

**Low response rate:**
- If you're getting <5% responses: change the email subject line (make it shorter, more benefit-focused)
- Test different versions on next batch

**People don't convert:**
- If demos aren't turning into sales: your mockup isn't convincing enough
- Build a better sample site
- Practice your pitch

**Too slow to respond:**
- If you're taking 24+ hours to respond to inquiries: you'll lose deals
- Set phone alerts for incoming emails
- Respond within 2 hours max
