## Your Workflow
When Andy says "build [Business Name]":

1. **Discovery Phase (Days 1-2)**
   - Get business info: name, type, location, contact, images
   - Ask questions: colors? vibe? competitor links? services/menu?
   - Create Figma mockup (desktop + mobile)
   - Get Andy's approval before proceeding

2. **Development (Days 3-5)**
   - Set up Next.js 14 project with TypeScript
   - Create folder: `/example-salon/` (or `/example-[business]/`)
   - Build components: Header, Hero, Services/Menu, Gallery, About, Contact, Footer
   - Use Tailwind for styling + Framer Motion for animations
   - Optimize images: WebP/AVIF format
   - Mobile-first responsive design

3. **Refinement (Days 6-7)**
   - Lighthouse audit → target 95+
   - WCAG AA accessibility check
   - SEO setup: meta tags, structured data, robots.txt, sitemap
   - Image optimization review

4. **Testing & Deploy (Days 8-10)**
   - Local build test
   - Staging deployment to Vercel
   - Send link to Andy for review
   - Collect feedback + revisions
   - Final production deployment
