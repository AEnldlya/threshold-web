## THE SOLUTION

---

## VERIFICATION CHECKLIST (Must pass ALL checks)

For each business, I must:

### ✓ Check 1: Google Search
```
Search: "[Business Name] [City] website"
```
- Does a website appear in results?
- If YES → SKIP this business
- If NO → continue

### ✓ Check 2: Google Business Profile
```
Search: "[Business Name] [City]"
Look at Google Business Profile card
```
- Does it list a website URL?
- If YES (even if clickable) → SKIP
- If NO → continue

### ✓ Check 3: Facebook
```
Search: "[Business Name] Boston" on Facebook
Click on their page
```
- Is there a "Website" link on their page?
- If YES → SKIP
- If NO → continue

### ✓ Check 4: Instagram
```
Search: "[Business Name]" on Instagram
Click on their profile
```
- Is there a link in bio?
- If YES → SKIP
- If NO → continue

### ✓ Check 5: Direct Domain Check
```
Try typing: [businessname].com in browser
Try: [businessname].net
Try: [businessname]boston.com
```
- Does any domain load a real website?
- If YES → SKIP
- If NO → continue

### ✓ Check 6: Yelp
```
Search: "[Business Name] Boston" on Yelp
```
- Is there a website link?
- If YES → SKIP
- If NO → PASS

---

## ONLY THEN

After ALL 6 checks pass with NO website found:
- Add business to call list
- Send to user

---

## WHAT "NO WEBSITE" MEANS

✗ They have a Facebook page (that's not a website)
✗ They're on Google Business with no external link (not a website)
✓ No custom domain
✓ No external website link anywhere
✓ Social media only, no dedicated site

---

## EXAMPLES

**SKIP (has website):**
- Restaurant with menu site
- Salon with booking calendar on own domain
- Plumber with service area map
- Any custom domain they own

**CALL (no website):**
- Business with Facebook page only
- Business on Google My Business with no website URL
- Instagram only, no link in bio
- Yelp listing but no external site
