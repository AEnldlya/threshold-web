# Website Build Checklist (3-Day Sprint)

Use this for every single project. Don't skip steps.

---

## DAY 1: KICKOFF & MOCKUP (4-5 hours)

### Morning (30 min)
- [ ] Kick-off call with client (15 min)
  - [ ] Confirm scope (what pages, services, etc.)
  - [ ] Get all content/photos
  - [ ] Set expectations (timeline, revisions)
  - [ ] Answer questions

### Mid-Morning (2 hours)
- [ ] Gather all content into folder
  - [ ] Create Google Drive folder: [Client Name] - Website
  - [ ] Ask them to upload photos/content
  - [ ] Download any photos they send
  - [ ] Extract Google/Yelp reviews if needed

- [ ] Create design brief
  - [ ] Write: Business name, tagline, main message
  - [ ] Note: Colors (if preference), tone, style
  - [ ] List: Main sections needed
  - [ ] Note: Any special requests

### Afternoon (2-3 hours)
- [ ] Create homepage mockup
  - [ ] Start with v4/v5 template as base
  - [ ] Update hero with their business info
  - [ ] Add their logo/colors
  - [ ] Update business info (phone, address)
  - [ ] Add their photos to hero
  - [ ] Draft 3-4 main sections
  - [ ] Add CTA buttons

- [ ] Share mockup for feedback
  - [ ] Send homepage mockup to client
  - [ ] Request feedback by EOD
  - [ ] "Anything you'd change?"

### Evening
- [ ] Review feedback
- [ ] Make notes on changes
- [ ] Plan Day 2

---

## DAY 2: BUILD & REFINE (5-6 hours)

### Morning (3 hours)
- [ ] Build remaining pages
  - [ ] Services page
  - [ ] About page
  - [ ] Contact page
  - [ ] (Optional 5th page if needed)

- [ ] Add content to all pages
  - [ ] Copy business info to each page
  - [ ] Add service descriptions
  - [ ] Add about/bio content
  - [ ] Add testimonials/reviews
  - [ ] Add photos throughout

- [ ] Set up functionality
  - [ ] Contact form (test it)
  - [ ] Google Maps embed (if applicable)
  - [ ] Phone number links (tel: format)
  - [ ] Email links (mailto: format)
  - [ ] Social media links (if applicable)

### Afternoon (2-3 hours)
- [ ] Design polish
  - [ ] Check colors/branding consistency
  - [ ] Adjust spacing if needed
  - [ ] Ensure images are properly sized
  - [ ] Check font sizes/readability
  - [ ] Mobile view — scroll through on phone

- [ ] Request second round of feedback
  - [ ] Screenshot full site (desktop + mobile)
  - [ ] Send to client
  - [ ] Ask for specific feedback on:
    - [ ] Colors/design
    - [ ] Content accuracy
    - [ ] Photos placement
    - [ ] Missing anything?

---

## DAY 3: FINAL & LAUNCH (3-4 hours)

### Morning (1-2 hours)
- [ ] Incorporate feedback
  - [ ] Make all requested changes
  - [ ] Update copy if needed
  - [ ] Adjust design as requested
  - [ ] Re-check mobile view

- [ ] Final quality check
  - [ ] All pages load (test each)
  - [ ] Contact form works (send test)
  - [ ] Links work (click every one)
  - [ ] Phone numbers are clickable
  - [ ] Images load properly
  - [ ] Spacing/alignment looks good
  - [ ] No typos (read every page carefully)
  - [ ] Mobile version looks good (scroll on phone)

### Afternoon (2-3 hours)
- [ ] Set up SEO basics
  - [ ] Page titles (unique for each page)
  - [ ] Meta descriptions (2-3 sentence summaries)
  - [ ] Alt text on images
  - [ ] H1 heading on each page
  - [ ] H2/H3 hierarchy correct

- [ ] Set up analytics
  - [ ] Google Analytics account
  - [ ] Add GA tracking code
  - [ ] Create account in Google Search Console
  - [ ] Submit sitemap

- [ ] Final checks
  - [ ] HTTPS/SSL enabled (automatic on Netlify)
  - [ ] Loading speed < 3 seconds
  - [ ] No broken links
  - [ ] Forms submission working
  - [ ] All contact info correct

- [ ] Deploy/Go live
  - [ ] Push to Netlify (or your hosting)
  - [ ] Test live URL works
  - [ ] Share with client
  - [ ] Celebrate!

### Evening
- [ ] Get final payment
  - [ ] Send invoice
  - [ ] Log into their payment
  - [ ] Confirm funds received

- [ ] Set up support
  - [ ] Give them hosting login (if needed)
  - [ ] Share domain details
  - [ ] Brief walkthrough (15 min call)
  - [ ] Send support contact info

---

## POST-LAUNCH (Week 1-2)

### Day 4-5
- [ ] Monitor for issues
  - [ ] Check that all pages load
  - [ ] Check contact form submissions
  - [ ] Google Analytics working

- [ ] First tweaks
  - [ ] Client may request small changes
  - [ ] Make notes (included vs. requires fee)
  - [ ] Implement included changes

### Week 2
- [ ] Set up Google My Business
  - [ ] Claim/create business listing
  - [ ] Add website URL
  - [ ] Add hours, phone, photos
  - [ ] Encourage reviews

- [ ] Submit to search engines
  - [ ] Google Search Console: submit sitemap
  - [ ] Bing Webmaster Tools: submit sitemap

- [ ] Offer upsell
  - [ ] After 2 weeks: "How's the website?"
  - [ ] Pitch monthly maintenance ($40/month)
  - [ ] Pitch premium features ($100-200)

## COMMON ISSUES & FIXES

| Issue | Fix |
|-------|-----|
| Contact form not working | Check Netlify forms is enabled; test submission |
| Images look blurry | Check image size (use 1200px wide minimum) |
| Page loads slowly | Compress images; remove unused CSS |
| Layout breaks on mobile | Check CSS media queries; test on phone |
| Styling looks wrong | Clear browser cache (Ctrl+Shift+Del); reload |
| Form not sending | Check spam folder; verify email in Netlify |
| Google Maps not showing | Check API key; reload page |
| Phone link not working | Check format: `tel:+15551234567` |

## TEMPLATE SITE BASE

**Always start with:**
- v4 or v5 template as your foundation
- Strip out the placeholder content
- Add client's business name + logo
- Update colors
- Customize copy

This cuts your build time from 3 hours → 2 hours.

**Time to create base template:** 1 hour (do once, reuse forever)
**Time to customize per client:** 2-3 hours
