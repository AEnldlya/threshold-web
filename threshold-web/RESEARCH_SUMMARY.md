### 3. GUIDE_3_ANIMATIONS_INTERACTIONS.md
**What you learned:**
- Pattern #1: Smooth hover effects (0.3s transitions, slightly darker/bigger)
- Pattern #2: Fade-in on scroll (elements appear as user scrolls)
- Pattern #3: Slide-in from side (smooth, professional)
- Pattern #4: Smooth scroll behavior (link clicks → smooth page scroll)
- Pattern #5: Button press feedback (briefly shrinks on click)
- Pattern #6: Hamburger menu animation (lines rotate to X)
- Pattern #7: Loading skeleton (gray placeholder shapes while content loads)
- Pattern #8: Form input focus (border color change + shadow)
- Pattern #9: Smooth color change (card hover effects)
- Pattern #10: Text counter animation (numbers animate from 0 to target)

- Good animation: Subtle, purposeful, under 0.3 seconds
- Bad animation: Spinning logos, bouncing text, auto-playing video

**What Actually Works:**
✓ Hover effects on buttons  
✓ Form focus effects  
✓ Smooth scroll  
✓ Fade-in on scroll  

**What Kills Conversions:**
❌ Auto-playing animations  
❌ Slow transitions

## What Works Across ALL Categories

**1. Hero Section**
- Clear, large headline

- Professional image or color background

**2. Trust Section**
- 5+ real customer reviews with names

**3. Services/Products**
- Organized by category

**4. Social Proof**
- Customer photos (if possible)
- Real names + locations

- Time-based proof ("trusted for 20 years")

**5. Booking/Contact**
- Multiple options (phone, form, email, text)
- Simple form (max 5 fields)
- Clear CTA button
- Instant confirmation

**6. Footer**
- Phone number (clickable on mobile)
- Address
- Hours
- Social links

## Design Rules That Never Fail

### Colors
- Pick ONE primary brand color
- Use white/light gray for background
- One accent color for highlights
- Test for accessibility (readable contrast)

### Fonts
- System fonts (Arial, Helvetica, Georgia) = faster + more reliable
- Font-size minimum 14px for body text
- Line-height 1.5-1.6 for readability
- Bold for headlines, regular for body

### Layout
- Max-width: 1200px (don't make lines too long)
- Padding: 20px sides (mobile), 40-60px (desktop)
- Grid-based (even spacing)
- Whitespace is professional

### Mobile
- Stack everything vertical on small screens
- Buttons full-width on mobile
- Phone number always clickable
- No horizontal scrolling

### Performance
- Images optimized (compress before upload)
- Load time target: <3 seconds
- No auto-playing video
- Minimal animations

## What to Avoid

❌ **Auto-playing video** — Kills conversions, slow loading  
❌ **Spinning logos** — Annoying, looks unprofessional  
❌ **Too many colors** — Looks cheap  
❌ **Small fonts** — Unreadable on phones  
❌ **Hidden contact info** — People give up  
❌ **"Call for quote"** — Customers bounce  
❌ **Horizontal scrolling** — Confuses users  
❌ **Slow animations** — Feels clunky  
❌ **Stock photos of people** — Obviously fake  
❌ **Long paragraphs** — No one reads them
